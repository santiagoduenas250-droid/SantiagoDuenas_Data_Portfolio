      </h3>
      <p align="center">
        <em>Python · PostgreSQL · scikit-learn · Claude API</em>
      </p>
      <p>
        An end to end churn analysis on <strong>6.7M KKBox members</strong> and <strong>30M listening events</strong>, paired with an automated monthly brief pipeline. Logistic regression (AUC 0.9665) surfaces the strongest churn levers, and a Claude API pipeline turns each month's KPIs and anomalies into a stakeholder ready brief without analyst drafting time.
      </p>
